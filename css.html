CSS

CSS is the styling language used to style HTML content. Adds visual appeal and presentation to a website/web-application.

CSS PROPERTIES

CSS uses a property name and value syntax (property name: property value;).

  • Property name specifies what to style of the targeted HTML elements.
  • Property value is the style applied to the HTML elements.

TYPES OF PROPERTIES

  • Color
  • Display
  • Font

Color

Color is used to change the color other of an HTML element.

Examples

  • Background Color
  • Text Color
  • Border Color

Background Color

The background-color property changes the color of the background. E.g. (background-color: red;)

Text Color

The color property changes the color of the text. E.g. (color: blue;)

Border Color

The border-color property changes the color of the border. E.g. (border-color: black;)
