HTML

HTML is the mark-up language used to create websites/web-applications. Should not be confused with a programming language.

HTML ELEMENTS

HTML uses a name value syntax (name=value).

  • The value attribute in HTML is used to specify the value of the element with which it is used.
  • The name attribute in HTML is used to specify the name of the element with which it is used.

TYPES OF ELEMENTS

  • Forms & Input
  • Audio & Video
  • Links
  • Lists
  • Tables
  • Styles & Semantics

Forms & Input

Forms are used to create an HTML Form for user input.

Examples

  • Input
  • Button
  • Option

Input

The input tag specifies an input field where the user can enter data. E.g. (input type="password")

Button

The button tag defines a clickable button. E.g. (button type="submit")

Option

The option tag defines an option in a select list. E.g. (option value="Edge")
